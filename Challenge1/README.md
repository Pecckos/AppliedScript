This Bash script collects basic system and environment information from a Linux system for reconnaissance and analysis purposes.

## What it does

The script gathers information about:

System and OS details

Current and logged-in users

Hardware and memory

Disks and storage

Network configuration and routing

Listening ports and active services

The script is read-only and does not modify the system.

## Usage

Make the script executable:

chmod +x InfoGatheringLinux.sh


Run it:

./InfoGatheringLinux.sh

## Purpose

Intended for educational use, lab environments, and basic security assessment.
